# Milestone Roadmap

## M0 — Foundation

Definition of done:

- RunPod SSH verified.
- Monorepo initialized with pnpm + uv.
- Shared schema package exists.
- Orchestrator emits tick over WebSocket.
- Browser displays tick counter.
- vLLM endpoint smoke test passes or fallback is documented.

## M1 — City Scene

- Procedural grid city.
- R3F scene with camera controls.
- Day/night cycle placeholder.
- Buildings/citizens/vehicles placeholders.

## M2 — Citizen Agents

- 20 personas generated or fixture-loaded.
- Memory stream + retrieval score.
- Plan tree.
- Dialogue event and memory panel.

## M3 — Vehicles & Vision

- 5 vehicles + 2 drones.
- Road graph + path following.
- Vision service `/detect`.
- Vehicle camera panel with boxes.

## M4 — Traffic AI

- Traffic env.
- Baseline signal controller.
- PPO wrapper.
- LSTM forecast payload and chart.

## M5 — God Mode

- Text command.
- Weather/event/person/infra/relationship handlers.
- Voice STT when stable.

## M6 — Polish & Demo

- Four-hour load test.
- Replay fallback.
- Demo script and metrics.
- Final README/GIF/slides support.
